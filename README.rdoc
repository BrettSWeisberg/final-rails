## Project Name: final-rails

## Project Overview:
-This app is the API back-end for a React app.

## Installation:
1. Fork and clone the repo
2. In the terminal type 'rails s -p 3001'
3. rake db:migrate
4. rake db:seed
5. rails s

## Tests:
There are currently no tests within the repo. All testing was done in the browser.
